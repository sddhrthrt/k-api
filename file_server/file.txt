# Text file serving system

## Deliverable

* Build a CLI with a `deploy` command that takes a text file URL and an endpoint path, and returns an endpoint to a live API
* An API takes a query parameter "index" and returns the word from the text file at that index, or an error if there is no word at the given index
* The APIs run in a Kubernetes cluster

## Example

```yaml
# https://my-bucket.s3.amazonaws.com/test.txt

This is a test file.
```

```text
$ cli deploy --text=https://my-bucket.s3.amazonaws.com/test.txt --path=test
endpoint: http://***/test
$ curl http://***/test?index=0
This
$ curl http://***/test?index=3
test
```

## Optional

* Implement a `list` command to show a list of live APIs
* Implement a `delete` command to delete an API
* Deploy to GCP or AWS
